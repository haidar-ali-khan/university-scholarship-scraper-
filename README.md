# University Scholarship Scraper

This project automatically scrapes the latest scholarship news
from a university website and saves it into a JSON file.

## Tech Stack
- Python
- BeautifulSoup
- Crawl4AI

## Features
- Daily scholarship data scraping
- Clean structured JSON output

## Use Case
Helps students stay updated with latest scholarship opportunities.
